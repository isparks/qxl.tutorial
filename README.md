## Developing DemoBrowser
Clone this repo and compile it:

```bash
    git clone https://github.com/qooxdoo/qxl.tutorial
    cd qxl.tuturial
    npm install @qooxdoo/compiler
    npx qx serve
```
Then open [http://localhost:8080](http://localhost:8080)
